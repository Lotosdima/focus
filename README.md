# Лендинг репетитора по математике

Одностраничный сайт для репетитора, специализирующегося на подготовке учеников 10-11 классов по математике к ЕГЭ.

## Структура сайта

Сайт построен по формуле ПИСДОС и включает следующие секции:

1. **Hero-секция** - Главный заголовок с призывом к действию
2. **Решение проблем** - Адресация болей и страхов аудитории
3. **О репетиторе** - Преимущества и достижения
4. **Программы обучения** - Три тарифа (Базовая, Профильная, Интенсив)
5. **Результаты** - Статистика и графики успеваемости
6. **Отзывы** - Реальные истории успеха
7. **Как проходит обучение** - Пошаговый процесс
8. **Гарантии** - Преимущества выбора
9. **Контакты и форма** - Запись на консультацию
10. **Футер** - Дополнительная информация

## Использование

Просто откройте файл `index.html` в браузере. Все стили подключены через CDN (Tailwind CSS и Font Awesome).

## Формула ПИСДОС в заголовке

- **П** - Профит: Гарантированный результат 80+ баллов
- **И** - Интрижка: Уникальная методика без скучной теории
- **С** - Срочность: Первые результаты уже через 2 недели
- **Д** - Доступность: Простое объяснение сложных тем
- **О** - Ограниченность: Только 6 месяцев до результата
- **С** - Сопричастность: Только для учеников 10-11 классов

## Особенности

- Полностью адаптивный дизайн
- Использование только HTML и Tailwind CSS (без JavaScript-фреймворков)
- Оптимизация для SEO
- Семантическая разметка
- Визуальная иерархия и четкие призывы к действию
